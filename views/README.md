# Task Manager Application

This application allows users to add, update, delete and save tasks 

#### Instructions
1. Download the source code
```
git clone [url to github repo]

```
or just download the *.zip file

2. Install deps

```
npm install
```
3. Run the app
``` 
node app
```

#### Deps list
- express
- pug
